Instalação da extensão do neovim no Visual Studio Code (VSCode)

  • Passo 1. Primeiro, abra o Visual Studio Code
  • Passo 2. Para navegar a aba de extensões, pressione CONTROL + SHIFT + X. Caso não abra a tela, feche todos os arquivos atuais pressionando CONTROL + SHIFT + F4 e depois pressione CONTROL + SHIFT + X novamente.
  • Passo 3. Pesquise pela extensão chamada "vim". É o primeiro que aparece na pesquisa, sua descrição é "Vim emulation for Visual Studio Code".
  • Passo 4. Instale a extensão e reinicie o VSCode.
  • Passo 5. Quando você reiniciar o VSCode, a extensão estará ativa. Agora, precisamos configurar ela. Pressione "CONTROL + ," para abrir a página de configuração do VSCode
  • Passo 6. Para entrar nas configurações da extensão do Vim, digite @ext:vscodevim.vim e aperte Enter
  • Passo 7. Navegue pressionando TAB até onde estiver escrito "Vim: Override Copy". Ative esta opção. Isto vai fazer com que você possa colar códigos de fora para dentro do VSCode.
  • Passo 8. Após ativar a opção, você pode pressionar CONTROL + W para fechar a página de configuração do VSCode, em seguida, pressione CONTROL + SHIFT + E para voltar ao explorador do VSCode.
  • Passo 9. Crie um arquivo para seguir os próximos passos do tutorial pressionando CONTROL + N.

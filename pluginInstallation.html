Instalação do extensão do neovim no Visual Studio Code (VSCode)

  1. Primeiro, abra o Visual Studio Code
  2. Para navegar a aba de extensãos, pressione CONTROL + SHIFT + X. Caso não abra a tela, feche todos os arquivos atuais pressionando CONTROL + SHIFT + F4 e depois pressione CONTROL + SHIFT + X novamente.
  3. Pesquise o extensão chamado "vim". É o primeiro que aparece na pesquisa, sua descrição é "Vim emulation for Visual Studio Code".
  4. Instale o extensão e reinicie o VSCode.
  5. Quando você reiniciar o VSCode, o extensão estará ativo. Agora, precisamos configurar ele. Pressione CONTROL + , (virgula) para abrir a página de configuração do VSCode
  6. Para entrar nas configurações do extensão do Vim, digite o seguinte e aperte Enter em seguida: @ext:vscodevim.vim
  7. Navegue pressionando TAB até onde estiver escrito "Vim: Override Copy". Ative esta opção. Isto vai fazer com que você possa colar códigos de fora para dentro do VSCode.
  8. Após ativar a opção, você pode pressionar CONTROL + W para fechar a página de configuração do VSCode, em seguida, pressione CONTROL + SHIFT + E para voltar ao explorer do VSCode.
  9. Crie um arquivo para seguir os próximos passos do tutorial pressionando CONTROL + N.

Comandos para a movimentação do cursor

  • h - mover o cursor para a esquerda.
  • j - mover o cursor para baixo.
  • k - mover o cursor para cima.
  • l - mover o cursor para a direita.
  • gj - mover o cursor para baixo (em texto de várias linhas).
  • gk - mover o cursor para cima (em texto de várias linhas).
  • H - mover para o topo da tela.
  • M - mover para o meio da tela.
  • L - mover para a parte inferior da tela.
  • w - avançar para o início de uma palavra.
  • W - avançar para o início de uma palavra (palavras podem conter pontuação).
  • e - avançar para o final de uma palavra.
  • E - avançar para o final de uma palavra (palavras podem conter pontuação).
  • b - retroceder para o início de uma palavra.

Comandos para movimentação parte 1

  • Lembrando que você utiliza as teclas H, J, K e L em vez das setas do teclado para mover o cursor, mas lembre-se que para se movimentar desta maneira, você deve estar no modo NORMAL, então pressione a tecla ESC para sair do modo de INSERÇÃO, caso você esteja nele.
  • Exercício: Encontre o erro no código abaixo e o corrija, use as teclas H, J, K e L para se movimentar no código.
  • 
#include <stdio.h>

int main() {
    // printf() displays the string inside quotation
    printf("Hello world")
    return 0;
}

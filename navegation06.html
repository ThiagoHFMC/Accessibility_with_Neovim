Comandos para movimentação parte 7

  • A busca de texto é uma parte vital de qualquer editor de texto. No Vim, você pressiona / e digita o texto que está procurando. Você pode repetir a busca por ocorrências seguintes e anteriores com n e N, respectivamente. Para casos de uso avançados, é possível usar expressões regulares que ajudam a encontrar textos de formatos específicos (no Vim real). Vamos tentar uma busca simples de texto.
  • Exercicio: Procure por int e encontre as ocorrências subsequentes com n.
  • 
#include <stdio.h>

int soma(int a, int b) {
    return a + b;
}

int multiplicacao(int a, int b) {
    return a * b;
}

int main() {
    int num1 = 5;
    int num2 = 3;

    int resultado_soma = soma(num1, num2);
    printf("A soma de %d e %d é: %d\n", num1, num2, resultado_soma);

    int resultado_multiplicacao = multiplicacao(num1, num2);
    printf("O produto de %d e %d é: %d\n", num1, num2, resultado_multiplicacao);

    return 0;
}

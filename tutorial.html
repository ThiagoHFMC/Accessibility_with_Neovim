Modos básicos

  • O Vim tem dois modos básicos. Um é o modo inserir, no qual você escreve texto como se fosse em um editor de texto normal. O outro é o modo normal, que oferece maneiras eficientes de navegar e manipular o texto. Para mudar entre os modos, use a tecla Esc para o modo normal e a tecla i para o modo de inserção. Vamos tentar!
  • Ao contrário do editor de texto comum, você utiliza as teclas H, J, K e L em vez das setas do teclado para mover o cursor, mas lembre-se que para se movimentar desta maneira, você deve estar no modo NORMAL, então pressione a tecla ESC para sair do modo de INSERÇÃO, caso você esteja nele.
  • Exercício: Faça uma inserção pressionando a tecla i dentro da função printf() com um texto considerando o código abaixo.
  • #include <stdio.h>
int main() {
    printf("");
    return 0;
}

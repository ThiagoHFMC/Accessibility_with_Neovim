Modos básicos

  • O Vim tem dois modos básicos. Um é o modo |inserir|, no qual você escreve texto como se fosse em um editor de texto normal. O outro é o modo |normal|, que oferece maneiras eficientes de navegar e manipular o texto. A qualquer momento, você pode ver em qual modo está na barra de status, localizada no topo do editor. Para mudar entre os modos, use |Esc| para o modo normal e |i| para o modo de inserção. Vamos tentar!
  • Exercício: Faça uma inserção dentro da função printf() com um texto considerando o código abaixo.
  • int main() {
    // printf() displays the string inside quotation
    printf("");
    return 0;
}

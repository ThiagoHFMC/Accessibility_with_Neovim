Comandos para movimentação parte 4

  • Para encontrar e ir para a próxima (ou anterior) ocorrência de um caractere, use f e a tecla F, por exemplo, pressione fo encontra o próximo "o". Você pode combinar f com um número. Por exemplo, você pode encontrar a 3ª ocorrência de 'q' com 3fq
  • Em texto estruturado com parênteses ou colchetes, ( ou { ou [, use % para pular para o parêntese ou colchete correspondente.
  • Exercício: O código abaixo possui várias funções printf, utilize a tecla % para pular para o final do parênteses nessas funções assim que perceber que o parênteses foi aberto
  • 
#include <stdio.h>

int main() {
    int num1, num2, soma;

    printf("Digite o primeiro número: ");
    scanf("%d", &num1);

    printf("Digite o segundo número: ");
    scanf("%d", &num2);

    soma = num1 + num2;

    printf("A soma de %d e %d é: %d\n", num1, num2, soma);

    return 0;
}

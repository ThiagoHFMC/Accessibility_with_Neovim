Comandos para movimentação parte 2

  • Para navegar pelo texto em termos de palavras, você pode usar as teclas |w|, |b| e |e| (também W, B, E no Vim real). |w| move para o início da próxima palavra; |e| move para o fim da palavra; e |b| move para o início da palavra.
  • Para ir para o início de uma linha, pressione |0|. Para ir para o fim de uma linha, use |$|
  • Exercício: Altere o nome da função que foi chamada dentro da função "main" que foi chamada com o nome incorreto. Utilize as teclas |w|, |b| e |e| para se movimentar pelo texto. 
  • #include <stdio.h>

void minhaFuncao() {
    printf("Esta é minha função!\n");
}

int main() {
    midFuncao(); // Chamada da função com nome incorreto
    
    return 0;
}

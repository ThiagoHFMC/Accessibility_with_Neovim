Comandos para criar e editar arquivos

  • :e[dit] file - editar um arquivo em um novo buffer.
  • :bn[ext] - ir para o próximo buffer.
  • :bp[revious] - ir para o buffer anterior.
  • :bd[elete] - excluir um buffer (fechar um arquivo).
  • :b[uffer]# - ir para um buffer pelo índice #.
  • :b[uffer] file - ir para um buffer por arquivo.
  • :ls ou :buffers - listar todos os buffers abertos.
  • :sp[lit] file - abrir um arquivo em um novo buffer e dividir a janela horizontalmente.
  • :vs[plit] file - abrir um arquivo em um novo buffer e dividir a janela verticalmente.
  • :vert[ical] ba[ll] - editar todos os buffers como janelas verticais.
  • :tab ba[ll] - editar todos os buffers em abas.
  • Ctrl + ws - dividir a janela.
  • Ctrl + wv - dividir a janela verticalmente.
  • Ctrl + ww - alternar entre janelas.
  • Ctrl + wq - fechar uma janela.

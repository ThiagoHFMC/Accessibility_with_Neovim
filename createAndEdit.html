Comandos para criar e editar arquivos

  • :h[elp] keyword - abrir ajuda para a palavra-chave.
  • :sav[eas] file - salvar o arquivo como.
  • :clo[se] - fechar o painel atual.
  • :q[uit] - fechar o arquivo atual.
  • :qw[rite] - fechar e salvar o arquivo atual.
  • :ter[minal] - abrir uma janela do terminal.
  • K - abrir a página do manual (man page) para a palavra sob o cursor.

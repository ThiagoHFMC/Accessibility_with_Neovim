Comandos para movimentação parte 3

  • Mover-se dentro do texto não se limita a teclas individuais; você pode combinar teclas de movimento com um número. Por exemplo, 3w é o mesmo que pressionar w três vezes.
  • Você também pode encontrar próximas palavras com base em onde o cursor está. Por exemplo, você pode encontrar a próxima ocorrência da palavra sob o cursor com *, e a anterior com #.
  • Exercício: Corrija o nome da função na sua chamada dentro da função main, porém, agora utilize combinações de números e teclas para se movimentar pelo texto. 
  • #include <stdio.h>

void minhaFuncao() {
    printf("Esta é minha função!\n");
}

int main() {
    midFuncao(); // Chamada da função com nome incorreto
    
    return 0;
}

Comandos para deletar

  • d é o comando de deletar. Você pode combiná-lo com movimentos, por exemplo, dw deleta a primeira palavra à direita do cursor
  • Para repetir o comando anterior, basta pressionar .
  • Exercicio: Primeiro, remova duas palavras com d2w. Depois, remova o resto das palavras nesta linha com . Utilizando o mesmo código do exercício anterior.

#include <stdio.h>

int soma(int a, int b) {
    return a + b;
}

int multiplicacao(int a, int b) {
    return a * b;
}

int main() {
    int num1 = 5;
    int num2 = 3;

    int resultado_soma = soma(num1, num2);
    printf("A soma de %d e %d é: %d\n", num1, num2, resultado_soma);

    int resultado_multiplicacao = multiplicacao(num1, num2);
    printf("O produto de %d e %d é: %d\n", num1, num2, resultado_multiplicacao);

    return 0;
}

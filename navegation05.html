Comandos para movimentação parte 6

  • gg te leva para o início do arquivo; G para o final.
  • Para ir diretamente para uma linha específica, digite seu número de linha junto com G.
  • Exercicio: vá para o início deste código com gg e depois volte para o final com G.
  • 
#include <stdio.h>

int main() {

    printf("Vá para o início do arquivo e depois volte para o final.");

    return 0;
}
